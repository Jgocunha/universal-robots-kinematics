Detailed Description

The eight inverse-kinematics solutions, each a full joint vector. valid[i] is true when solution i is geometrically feasible (no NaN angles). anyValid() returns true when at least one solution exists. For invalid rows, angle values are NaN — old-style consumers checking std::isnan() continue to work; valid[] is the authoritative flag.
